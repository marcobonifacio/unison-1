withRepo :: forall m a. MonadUnliftIO m => ReadGitRepo -> GitBranchBehavior -> (GitRepo -> m a) -> m (Either GitProtocolError a) Source #

Clone or fetch an updated copy of the provided repository and check out the expected ref, then provide the action with a path to the codebase in that repository. Note that the repository provided to the action is temporary, it will be removed when the action completes or fails.
